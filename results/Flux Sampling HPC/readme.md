# Flux Sampling HPC

## Overview

This directory contains the flux sampling results generated from the context-specific metabolic models using ACHR sampling (COBRA Toolbox).

The sampling files are **not included** in this repository because they exceed GitHub's file size limits.

They can be regenerated by executing the HPC sampling workflow described in `scripts/README.md`.

---

# Standard Cell-Line Models

Sampling results generated from the medium-constrained cell-line models are stored as:

```
samplingResults_medium_1000_model_1.mat
samplingResults_medium_1000_model_2.mat
...
samplingResults_medium_1000_model_8.mat
```

Model ordering:

| Model | Biological model |
|------:|------------------|
| 1 | 769-P Control |
| 2 | 769-P siAKR1A1_1 |
| 3 | 769-P siAKR1A1_12 (consensus) |
| 4 | 769-P siAKR1A1_2 |
| 5 | Huh7 Control |
| 6 | Huh7 siAKR1A1_1 |
| 7 | Huh7 siAKR1A1_12 (consensus) |
| 8 | Huh7 siAKR1A1_2 |

### Legacy filenames

The original workflow generated sampling files with 1500 samples:

```
SamplingResults_medium_1500_model_1.mat
...
SamplingResults_medium_1500_model_8.mat
```

These legacy filenames are retained for compatibility with the original publication workflow.

# TCGA AKR1A1 ON/OFF Models

Sampling results generated from the medium-constrained TCGA models are stored as:

```
samplingResults_mediumonoff_1000_model_1.mat
samplingResults_mediumonoff_1000_model_2.mat
...
samplingResults_mediumonoff_1000_model_8.mat
```

The numbering follows the ordering of the models stored in:

```
models_mediumonoff.mat
```

---

## GLO1 Knockout Sampling

For simulations including GLO1 knockout, the corresponding files are:

```
samplingResults_medium_KO_GLO1onoff_1000_model_1.mat
...
samplingResults_medium_KO_GLO1onoff_1000_model_8.mat
```

These files are used by the Flux Sum and Flux Sampling analyses when comparing medium and GLO1 knockout conditions.

---

# File Contents

Each sampling file contains:

- `x.samples` – sampled reaction flux distributions
- `x.modelSampling` – sampled metabolic model

These files serve as the input for the Flux Sum and Flux Sampling statistical analyses.

---

# Notes

Only the sampling output files are omitted from the repository due to their size.

The sampling parameters, model ordering, and downstream analyses are identical to those used in the published computational workflow.
