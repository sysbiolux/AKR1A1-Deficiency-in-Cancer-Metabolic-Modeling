# AKR1A1-Deficiency-in-Cancer-Metabolic-Modeling
Genome-scale metabolic modeling workflow investigating AKR1A1 deficiency as a metabolic vulnerability in renal cell carcinoma and hepatocellular carcinoma. Integrates cell-line and TCGA tumor RNA-seq data with rFASTCORMICS/Recon3D reconstruction, FBA, FVA, flux sampling, flux-sum analysis, and IDARE/Cytoscape visualization.
